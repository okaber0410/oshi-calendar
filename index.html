使い方・ヘルプ

プロファイルの作り方・切り替え方

📌 新しい推しを登録する
設定画面の「＋ 新規推し登録」ボタンから追加できます。推しごとに名前・カラー・バナー・記録がすべて独立して管理されます。

📌 推しを切り替える
画面上部のドロップダウン（▼）から切り替えられます。設定画面内の一覧からも切り替え可能です。

📌 プロファイルを削除する
設定画面のプロファイル一覧にあるゴミ箱アイコンから削除できます。最後の1人は削除できません。

推しカラー・バナーの変更

📌 推しカラーを変える
設定画面の「推し色設定」からカラーパレットまたはスライダーで色を選んで「確定」を押してください。アプリ全体の色が変わります。

📌 バナー画像を設定する
「バナー工房」ボタンから画像を選択できます。ズーム・位置調整も可能です。推しの公式画像やスクリーンショットをお使いください。

📌 テーマを変える
「ベーステーマ」から漆黒・深海・桜・琥珀の4種類を選べます。

記録の追加・編集・削除

📌 記録を追加する
カレンダーの日付をクリックしてサイドパネルを開き、「＋ 新しい記録を追加」を押します。タイトル・URL・タグを設定して保存してください。

📌 記録を編集する
サイドパネルで編集したい記録カードをクリックすると編集画面が開きます。

📌 記録を削除する
サイドパネルで記録カードにマウスを乗せるとゴミ箱アイコンが表示されます。クリックで削除できます。

📌 スケジュール同期をONにする
記録エディタの「スマホとスケジュール同期する」をONにすると、その記録がQRコードの対象になります。

タグの使い方

📌 タグを追加する
設定画面の「タグ管理」から名前と色を選んで追加できます。記録を分類するのに使います。

📌 タグで絞り込む
画面左のタグ一覧をクリックするとそのタグの記録だけが表示されます。「CLEAR」で解除できます。

📌 システムタグについて
「スケジュール」「OTHER」の2つは削除できない特別なタグです。スケジュールタグはQRコード連携に必須です。

APIキーの設定方法

APIキーを設定すると、推しのYouTube配信スケジュールを自動で取得できるようになります。取得は無料・登録不要です。

📌 設定手順の動画を見る

YouTube で設定方法を見る

⚠️ APIキーはあなたのブラウザにのみ保存されます。こちらには届きません。

QRコードでカレンダーに登録する

📌 QRコードを生成する
左のタグ一覧から「スケジュール」を選択 → ダッシュボードが表示 → 登録したい予定にチェックを入れて「チェックした予定をQR化」を押します。

📱 スマホのカレンダーへの反映方法

🟢 Googleカレンダー（Android・PC）

QRをカメラで読む → リンクをタップ → 「保存」を押すだけで自動登録されます。

🍎 iOSカレンダー（iPhone・iPad）

QRをカメラで読む → リンクをタップ → 「カレンダーに追加しますか？」が表示されるので「追加」を選択してください。

🔵 Outlook

QRをカメラで読む → .icsファイルをダウンロード → ファイルをダブルクリックするとOutlookが起動して登録できます。

⚠️ 一度に登録できるのは最大5件です。

バックアップ・復元の方法

📌 バックアップを保存する
設定画面の「📦 バックアップ」ボタンを押すとJSONファイルがダウンロードされます。定期的に保存しておくことをおすすめします。

📌 バックアップから復元する
設定画面の「📂 バックアップから復元する」ボタンから保存したJSONファイルを選択してください。

⚠️ 復元すると現在のデータは上書きされます。事前にバックアップを取ることをおすすめします。

📌 完全初期化する
設定画面最下部の「⚠️ システム完全初期化」からすべてのデータを削除できます。この操作は取り消せません。
